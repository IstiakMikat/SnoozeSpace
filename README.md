# SnoozeSpace

This repository contains the full SnoozeSpace project with both the client and server.

## Structure

- `SnoozeSpace-client/` - React/Vite frontend application
- `SnoozeSpace-Server/` - Node.js backend API

## Setup

### Client

```powershell
cd "g:\cse 470 project\SnoozeSpace-client"
npm install
npm run dev
```

### Server

```powershell
cd "g:\cse 470 project\SnoozeSpace-Server"
npm install
node index.js
```
